# Sealink Twig
Sealink Craft Plugin for Twig Extensions

Adds common twig methods that are shared across CMS sites.

## Installation
Install using composer

```
  "require": {
    "sealink/twiglink": "0.1.0"
  }
```

## Tests
composer install
phpunit
